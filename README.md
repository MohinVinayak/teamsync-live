# TeamSync Live — Real-Time Chat Application using DevOps Pipeline

TeamSync Live is a simple real-time chat application built with Node.js and Socket.IO. The project demonstrates a complete DevOps workflow including CI/CD automation, containerization, cloud deployment, and monitoring.

---

## Tech Stack

### Application
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript

### DevOps Tools Used
- GitHub – Version control and collaboration
- Docker – Containerization
- Docker Hub – Container registry
- GitHub Actions – CI/CD automation
- Render – Cloud deployment
- UptimeRobot – Monitoring and alerting

---

## Features

- Real-time chat communication using WebSockets
- Automatically deployed via CI/CD pipeline on each commit
- Hosted on cloud with uptime monitoring
- Lightweight and scalable
